Teiler

Little script for screenshots and screencasts utilizing rofi, maim, ffmpeg

View project on GitHub

Features

  • Create Screenshots and Screencasts
    • from screen(s)
    • from area
    • with delay
  • Upload
    • to your own ssh server
    • to a filebin server
    • to imgur
  • Browse History and
    • view images/videos
    • edit images and
    • re-upload
### Dependencies

* [rofi](https://github.com/DaveDavenport/rofi)
* [maim](https://github.com/naelstrof/maim)
* [slop](https://github.com/naelstrof/slop)
* [ffmpeg](https://github.com/FFmpeg/FFmpeg)
* [copyq](https://github.com/hluk/CopyQ)

### Nice to have:

* A Notification Daemon (Preferably [dunst](https://github.com/knopwob/dunst)
* [dzen2](https://github.com/robm/dzen) with Xinerama Support

Installation

  • Make sure to have all Dependencies installed.
  • Clone Repository
  • Copy config.example to $HOME/.config/teiler/config
  • Tweak it to your liking
  • Enjoy
`teiler --help` for more information about possible hotkeys
