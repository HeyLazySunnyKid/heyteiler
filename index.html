Teiler

Little script for screenshots and screencasts utilizing rofi, maim, ffmpeg

View project on GitHub

Features

  • Create Screenshots and Screencasts
    • from screen(s)
    • from area
    • with delay
  • Upload
    • to your own ssh server
    • to a filebin server
    • to imgur
  • Browse History and
    • view images/videos
    • edit images and
    • re-upload
